# Python Week 2 Assignments

This repository contains solutions for Week 2 Python assignments.

## Assignments Included

### 1. List CRUD Operations
- Create elements in a list
- Read list data
- Update list elements
- Delete list elements

### 2. Word Frequency Counter
- Takes user input
- Counts frequency of each word using a dictionary

### 3. Pattern Printing
- Uses nested loops
- Prints star patterns

### 4. Largest and Smallest Number Finder
- Finds maximum and minimum values from a list

### 5. Sorting Using Lambda Function
- Sorts data using Python's lambda expressions

### 6. Contact Book Using Dictionary
Features:
- Add Contact
- Search Contact
- Update Contact
- Delete Contact

### 7. Word Counter From Text File
Features:
- Count total words
- Count total characters
- Count total lines

### 8. JSON File Reader
Features:
- Load JSON data
- Display formatted JSON output

## Mini Project: Student Management System

### Features

- Delete Student
- Store data in CSV file

### Technologies Used
- Python
- CSV Module
- JSON Module

## How to Run

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Open the project folder:

```bash
cd project-folder
```

3. Run the Python file:

```bash
python filename.py
```
